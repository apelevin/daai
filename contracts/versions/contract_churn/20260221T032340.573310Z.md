# Data Contract: Contract Churn

## Статус
Согласован

## Определение
Коммерческий отток клиентов — количество клиентов, которые не продлили или расторгли договор в отчётном периоде.

## Формула
```
Contract Churn Rate = (Количество клиентов, не продливших договор в периоде) / (Общее количество клиентов на начало периода) × 100%
```

## Источник данных
CRM/контрактный реестр. Поля: дата окончания договора, статус продления, причина непродления.

## Включает
- Непродление договора по окончании срока
- Досрочное расторжение по инициативе клиента
- Досрочное расторжение по инициативе компании

## Исключения
- Задержка оплаты по согласованию (grace period) — НЕ churn
- Паузы/заморозки по соглашению сторон
- Миграции между юрлицами (тот же клиент)
- Технические отмены/переоформления

## Гранулярность
Месячный расчёт. Обновление еженедельно.

## Связь с Extra Time
Contract Churn → снижение Retention → уменьшение MAU → падение Extra Time

## Потребители
- Sales: прогноз, retention-анализ
- Product: health score клиентов
- Finance: планирование выручки

## Состояние данных
- Данные есть, качество подтверждено: CRM/контрактный реестр
- Данных нет: автоматизированные алерты о непродлении

## Известные проблемы
Нет автоматических уведомлений о приближающихся непродлениях.
