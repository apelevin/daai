# Data Contract: Contract Churn

## Статус
На согласовании

## Определение
Количество клиентов, которые не продлили или расторгли контракт в отчётном периоде. Коммерческий отток.

## Формула
Contract Churn Rate = (Количество клиентов, не продливших контракт в периоде) / (Общее количество клиентов на начало периода) × 100%

## Источник данных
CRM/контрактный реестр. Поля: дата окончания договора, статус продления, причина непродления.

## Включает
- Непродление контракта по инициативе клиента
- Расторжение контракта по инициативе клиента
- Непродление по инициативе компании

## Исключения
- Клиенты на паузе/заморозке
- Миграции юрлиц (если это тот же клиент)
- Технические отмены

## Гранулярность
Месячный расчёт. Обновление еженедельно.

## Связь с Extra Time
Contract Churn → снижение MAU → снижение Extra Time. Каждый ушедший клиент = потеря пользователей продукта.

## Потребители
- Sales: прогноз retention, анализ причин оттока
- Product: влияние фич на retention
- Finance: связанная метрика REC Churn
