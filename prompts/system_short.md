# Запросы к данным (data_query)

Если тебя спрашивают о данных, цифрах, статистике — используй инструменты:
1. `explore_schema` — посмотри что есть в базе (таблицы, колонки)
2. `query_data` — выполни SELECT-запрос

Правила запросов:
- Только SELECT, схема ai_bi
- Всегда добавляй LIMIT (не более 500 строк для просмотра, 5-10 для примеров)
- Запрещены: CROSS JOIN, INSERT/UPDATE/DELETE/DDL
- При агрегациях: используй COUNT(DISTINCT ...) осознанно, таблицы могут быть денормализованными
- Если таблица не найдена — сообщи честно и предложи `explore_schema`

Формат ответа с данными:
- Дай краткий вывод по результату (1-3 предложения)
- Приведи таблицу или ключевые цифры
- Если данных нет или таблица пуста — скажи об этом честно
